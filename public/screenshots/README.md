# Screenshots

Place your product screenshots here.

Expected files:
- `thermal-analysis.png` — Thermal Analysis Dashboard screenshot
- `glide-efficiency.png` — Glide Efficiency Report screenshot
- `coaching-summary.png` — Coaching Summary screenshot

These will appear in the "Product Preview" section on the home page.
Recommended dimensions: 1200×750px (16:10 ratio works well).

Replace the placeholder cards in `components/ScreenshotGallery.tsx`
by updating `PlaceholderScreen` to use `<Image>` from `next/image` once
your screenshots are ready.
